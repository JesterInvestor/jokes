# Sound Files

This directory contains sound effects for the Jokes app.

## Required Sound Files

Place the following MP3 files in this directory:
- `click.mp3` - Navigation click sound
- `vote.mp3` - Voting action sound  
- `submit.mp3` - Joke submission sound
- `share.mp3` - Share action sound

## Note

For now, the app will gracefully handle missing sound files.
Add your own sound effects to enhance the user experience!
